GAN

The network structure of GAN consists of generator network and discriminator network. The generator receives random variable and generates fake sample data . The goal of the generator is to make the generated sample as similar as possible to the real sample. The input of the discriminator consists of two parts, namely the real data and the data generated by the generator. The output of the discriminator is usually a probability value, representing the probability that considers the input to be a real distribution. If the input is from the real data, it outputs 1, otherwise it outputs 0. At the same time, the output of the discriminator will be fed back to to guide the training of . Ideally, cannot judge whether the input data comes from real data or generated data , that is, the output probability value of every time is 1/2 (equivalent to random guess), and then the model is optimal. In practice, the generator network and discriminator network are usually constructed by deep neural network.

The idea of GAN comes from the two-person zero-sum game in game theory, where the generator and discriminator can be viewed as two players in a game. In the process of model training, the generator and discriminator will update their own parameters to minimize the loss. A generator is essentially a differentiable function that takes input from a random variable and generates a fake sample . In GAN, the generator generally has no restrictions on the input variable , which is usually a 100-dimensional random encoding vector, and can be random noise or a variable that fits some distribution. Theoretically, the generator can gradually learn any probability distribution, and the trained generation network can generate a realistic image, but it is not the same as the real image. That is, the generation network learns an approximate distribution of the training data, which is particularly important in the application of data augmentation.

Like the generator, the discriminator is a differentiable function as well. In GAN, the main purpose of the discriminator is to judge whether the input is a real sample and provides feedback to guide the generator training. The discriminator and generator form two players in a zero-sum game. In order to win the game, the discriminator and generator constantly improve their discriminating ability and generating ability through training. The game will eventually reach a Nash equilibrium state, where the generator learns the probability distribution approximating to the real sample. The discriminator has been unable to correctly determine whether the input data came from the real sample, or the fake sample generated by the generator, that is, the discriminator output each time with a probability value of 1/2.
